# Tower Defence - Rogue-like Edition

A feature-rich HTML5 Tower Defense game with Rogue-like mechanics, continuous enemy generation, and extensive customization systems.

## Features

### Core Gameplay
- **Continuous Enemy Generation**: No wave-based system - enemies spawn continuously with increasing difficulty
- **20 Unique Towers**: Each with distinctive abilities, costs, and upgrade paths
- **30 Unique Enemies**: Different behaviors, resistances, and combat properties
- **50+ Power-ups**: Rogue-like run modifiers for infinite replayability

### Game Systems
- **Dynamic Difficulty**: Scaling enemy spawns and properties based on progression
- **Tower Placement**: Free placement with range indicators and cost management
- **Upgrade System**: Enhance towers with damage, range, and attack speed improvements
- **Asset Integration**: Uses game-icons.net-compatible assets and external sprite sourcing

### Rogue-like Elements
- **Power-up Selection**: Random power-ups offered after certain milestones
- **Permanent Upgrades**: Some power-ups provide run-wide stat bonuses
- **Synergy System**: Power-ups interact with towers and game mechanics
- **High Score Tracking**: Persistent leaderboard via localStorage
